# Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Sales & Revenue Analysis Dashboard developed using Power BI and Python. It helps analyze sales performance, profit, product performance, and regional trends using business intelligence techniques.

## 🎯 Objectives
- Analyze total sales and profit
- Track monthly sales trends
- Compare regional performance
- Identify top-performing products
- Provide interactive filtering using slicers

## 🛠️ Technologies Used
- Power BI
- Python
- Pandas
- Microsoft Excel / CSV
- GitHub

## 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Sales KPI
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top 10 Products
- Interactive Slicers (Region, Category, Segment)
